# Data-Science-Python

The purpose of this repository is to document my data science, machine learning, and deep learning projects using Python. The work showcased here ranges from foundational descriptive statistics and classical statistical modeling (e.g., multiple linear regression, logistic regression, and hierarchical models) to more advanced machine learning and deep learning applications.

Projects in this repository include supervised and unsupervised learning, model evaluation, feature engineering, and neural network architectures such as CNNs and other deep learning models. Many of these notebooks are part of ongoing academic research, capstone projects, and independent experiments focused on real-world prediction, computer vision, and educational data science.

This repository serves as a living portfolio of my progression from statistical analysis to full-scale machine learning and deep learning workflows.
